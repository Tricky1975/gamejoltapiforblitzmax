GameJolt.GJ: Types Modinfo Source  

This module contains the stuff you need to make your BlitzMax program connect to GameJolt

This module has been designed to take advantage of the Threaded mode, but as some programs can turn out a bit allergic to Threaded mode, I did enter a possibility to use this API unthreaded, however this will slow down your game when updates on GameJolt are done.

Types Summary

GJ Used to setup some data for GameJolt.
gjTable Features for fetching hi-scores.
gjTrophy Type which contains the data about a trophy.
gjUser Creates a user who can send his or her data to GameJolt.
gjUserInfo Fetches and stores the data about a user (not one who is logged in the game, just an other user. Did I explain this well?)
